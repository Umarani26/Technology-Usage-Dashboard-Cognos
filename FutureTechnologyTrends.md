# Future Technology Usage Dashboard 

This dashboard shows the technologies developers want to work with in the future.

It includes:
- Desired programming languages
- Future database preferences
- Cloud/platform technologies in demand
- Web frameworks developers want to learn

This helps identify upcoming technology trends in the industry.

## Key Insights
1. PostgreSQL is the most used database.
2. Microsoft Azure dominates platform usage.
3. ASP.NET Core is among the most popular web frameworks.
4. C# appears frequently in language combinations.

## Findings
Respondents are primarily interested in building skills around cloud computing, modern development stacks, and scalable technologies, with AWS emerging as the most desired platform.

## Implication:
Organizations and training providers should prioritize cloud, full-stack development, and modern programming skills, as these areas are likely to drive future technology adoption and workforce demand.
