# Current Technology Usage Dashboard 

This dashboard shows the technologies currently used by developers based on survey data.

It includes analysis of:
- Programming languages currently used
- Databases in use
- Cloud/platform services
- Web frameworks

This helps understand the current technology landscape in the developer community.

## Key Insights

Most developers work with C#, JavaScript, and SQL, showing strong full-stack demand. PostgreSQL, MySQL, and SQL Server dominate database usage, while AWS and Azure lead platform adoption. In frameworks, ASP.NET Core and React are most commonly used, showing a shift toward modern web development.

## Findings
- C#, JavaScript, and SQL are the most common language combinations.  
- PostgreSQL, MySQL, and SQL Server are the leading databases, with MongoDB also growing.  
- AWS, Azure, and ASP.NET Core are the most widely used platforms and frameworks.

## Implications
- Developers should strengthen full-stack skills (C#, JS, SQL).  
- Cloud expertise (AWS/Azure) and database skills are essential for job readiness.  
- Learning modern frameworks like ASP.NET Core and React improves career opportunities.
