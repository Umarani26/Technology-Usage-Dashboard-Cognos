# Technology Trends & Developer Survey Dashboard

## Overview

This project is a data visualization dashboard built using a developer survey dataset. It analyzes current technology usage, future technology preferences, and demographic distribution of respondents to identify key trends in the software industry.

The project is divided into three separate dashboard files for better organization and clarity.
- Current Technology Usage
- Future Technology Trends
- Demographics Analysis

## Current Technology Usage

This section shows the technologies developers are currently working with.

Includes:

- Top programming languages
- Most used databases
- Popular cloud/platform services
- Common web frameworks

Key insight: Most developers currently work with JavaScript, C#, SQL, AWS, Azure, and popular databases like MySQL and PostgreSQL.

View Dashboard 1: [Current Technology Trends](CurrentTechnologyTrend.md)

## Future Technology Trends

This section highlights what developers want to learn or work with in the future.

Includes:

- Desired programming languages
- Preferred databases
- Future platform preferences
- Web frameworks in demand

Key insight: Strong interest in Python, Rust, Kotlin, BigQuery, and modern cloud-native technologies.

View Dashboard 2: [Future Technology Trends](FutureTechnologyTrends.md)

## Demographics Analysis

This section provides insights into the survey participants.

Includes:

- Age distribution
- Country-wise response distribution
- Education level breakdown
- Age vs education comparison

Key insight: Most respondents are aged 25–34, with the majority holding bachelor’s or master’s degrees, and a large number of responses coming from the United States.

View Dashboard 3: [Demographics Analysis](Demographics.md)

The goal of this project is to understand current and future technology trends in the developer community and how demographics influence technology adoption.
